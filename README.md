# Monte-Carlo-S-P-500-Portfolio-Risk-Simulation
A Monte Carlo simulation model built in Excel to analyze portfolio risk and estimate the probability of achieving long-term investment targets under uncertain returns.
This project models a 30 year long term performance of a $10,000 S&P500 portfolio using the Monte Carlo simulation. By generating 1,000 stochastic return scenarios, the model evaluates the distribution of possible outcomes and quantifies both downside and upside potential.
**Key Features**
Simulates 1,000 portfolio scenarios using normally distributed returns
Analyzes outcome distribution with percentile metrics (5th, median, 95th)
Calculates probability of achieving a target portfolio value
Visualizes results using a dashboard with histograms
**Tools Used**
Microsoft Excel
Data Tables
Statistical functions
