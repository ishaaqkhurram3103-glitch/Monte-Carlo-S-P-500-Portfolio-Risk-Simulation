An Excel-based Monte Carlo simulation model designed to analyze long-term portfolio risk and estimate the probability of achieving investment targets under uncertain market returns.

---

## Project Overview

This project models the 30-year performance of a $10,000 investment in the S&P 500 using Monte Carlo simulation techniques in Microsoft Excel.

By generating 1,000 stochastic return scenarios, the model evaluates the distribution of possible portfolio outcomes and quantifies both downside risk and upside potential under varying market conditions.

The project focuses on understanding uncertainty, variability in long-term investing outcomes, and probability-based decision making.

---

## Key Features

- Simulates 1,000 portfolio scenarios using normally distributed annual returns
- Models long-term compounding growth over a 30-year horizon
- Analyzes outcome distribution using percentile metrics:
  - 5th Percentile (Downside Case)
  - 50th Percentile (Median Outcome)
  - 95th Percentile (Upside Case)
- Calculates probability of achieving a target portfolio value
- Visualizes simulation outcomes using dashboard metrics and histograms
- Tracks best-case, worst-case, and average simulated outcomes
